# Meta Crush Saga
A C++17 compile-time game

Thanks to [Alexandre Gordeev](https://www.linkedin.com/in/alexandre-gordeev/) who helped making the game experience even better!

## How to run the game:

+ Have GCC with a version > 7.x
+ Have bash
+ Call `./meta_crush_saga.sh`
